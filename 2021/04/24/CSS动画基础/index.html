transform属性：教程

transform 属性向元素应用 2D 或 3D 转换。该属性允许我们对元素进行旋转、缩放、移动或倾斜。

常见属性

  • translate translateX translateY x轴 y轴 定向移动 单位：百分比或者px
  • scaleX（x，y） 缩放比例 填倍数
  • rotate(angle) 旋转角度 rotate(60deg)
  • skew（angle） 倾斜角度
