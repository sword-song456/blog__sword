JavaScript变量理解

var&&let&&const

ECMAScript变量是松散类型的，也就意味这，变量可以保存任何类型的数据。每个变量只不过是一个用于保存任意值的命名占位符

var

  • 声明作用域为函数

  • var 声明提升

    所谓提升，就是将声明拉到函数作用域的顶部 （只拉声明 ，不拉赋值）

let

  • 块作用域 块 ===》 {}

  • 声明不会被提升

  • 可全局声明 ，但不会成为window对象的属性（var）

  • 不允许重复声明

const

  • 与let相似 唯一一个重要的区别是声明变量时必须要初始化，且不允许修改

生命风格以及最佳实践

  • 不使用 var

  • const 优先 let 次之

    使用const 可以使浏览器强制保持变量不变 ， 只在提前知道变量会修改时使用let来声明
