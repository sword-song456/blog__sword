一、知识储备

Hexo

  • 快速 、简洁且高效的博客框架，使用Markdown , 解析文章

  • 安装前提： node.js Git

    可想而知 ，hexo博客搭建过程以及后续的博客更新是基于git分布式版本控制系统、node.js

  • 文件目录

  • 在 _config.yml 中修改大部分的配置。同理于themes中的_config.yml 详情

  • 常用命令

  • 其他内容详情请看 官方文档

gitee / github

  • 相同点

    • 代码托管 协作开发的 软件项目托管平台
    • 开源软件仓库
    • 基于git
    • 支持静态网站服务 ⭐

  • 不同点

    • gitee 访问稳定 速度快 ，github访问不稳定 ⭐

    • github 相比于 gitee 用户丰富， 开源软件丰富 ， 库多
